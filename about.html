About This Project

Local Market Price Tracker waa website loo sameeyay in lagu diiwaangeliyo laguna la socdo qiimaha alaabaha suuqyada maxalliga ah.

Ujeeddada mashruucan waa in uu dadka ka caawiyo ogaanshaha isbedelka qiimaha badeecadaha maalin walba si fudud oo degdeg ah.

Website-kan waxaa lagu dhisay HTML, CSS iyo JavaScript oo keliya.
